### **🚀 Quick Setup Guide**  

#### **1️⃣ Set Up Virtual Environment**  
```bash
python -m venv venv  
source venv/bin/activate  # For Mac/Linux  
venv\Scripts\activate  # For Windows  
```

#### **2️⃣ Install Dependencies**  
```bash
pip install -r requirements.txt  
```

#### **3️⃣ Run the App**  
```bash

python3 main.py --port 8080 
```
